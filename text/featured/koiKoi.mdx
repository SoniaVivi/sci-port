# Koi Koi

**Recreation of the tradional Japanese card game**

Created with: Typescript, React, Redux, Next.js

Developed using a TDD (test-driven development) approach.

Proper game implementation is ensured with comprehensive unit tests utilizing Jest.

Features:

Helpful features to aid with learning how to play such as a yaku list and highlighting possible card matches

Multiple AI's using different implementation logic to help players play against AI's at their skill level.
