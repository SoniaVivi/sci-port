# Vivid Image Organizer

**Built For Handling Large Quantities Of Images Effectively**

Created with: Python, SQL, Kivy (Python GUI Framework)

**Introduction**

Born while working night shifts at my first job, Vivid is the first big project I tackled that provides real world utility. Designed solely by me with minimal use of frameworks and libraries.

**Featuring**

- A fully self-designed ORM, usable in other projects

- Capable of handling large quantities of images

- A command line interface

- Supports searching by name, tag, and image hash

- Blacklisting images and directories

- Opening images in a program of your choice

- Mass tagging and deleting

- Fully documented database Python SQLite interface

**Use the dev-version-2 branch version for the greatest and latest features!**
